13. Asynchronous Programming

Learn how JavaScript handles tasks over time

What is Asynchronous Programming?

Asynchronous programming allows JavaScript to perform long-running tasks (like API calls, timers, file loading) without blocking the main thread.

JavaScript is single-threaded, but it uses the Event Loop to handle asynchronous tasks efficiently.

Assignment Operators

Used to assign values to variables:

let x = 10;
x += 5; // Same as x = x + 5 (15)
x -= 2; // Same as x = x - 2 (13)
x *= 3; // Same as x = x * 3 (39)
x /= 3; // Same as x = x / 3 (13)
x %= 4; // Same as x = x % 4 (1)

setTimeout()

The setTimeout() function runs code after a specified delay.

console.log("Start");

setTimeout(() => {
  console.log("Executed after 2 seconds");
}, 2000);

console.log("End");

Promises

A Promise represents a value that may be available now, later, or never.

const myPromise = new Promise((resolve, reject) => {
  let success = true;

  if (success) {
    resolve("Operation Successful!");
  } else {
    reject("Operation Failed!");
  }
});

myPromise
  .then(result => console.log(result))
  .catch(error => console.log(error));

Async & Await

async makes a function return a Promise. await pauses execution until the Promise resolves.

function delay() {
  return new Promise(resolve => {
    setTimeout(() => resolve("Done!"), 2000);
  });
}

async function run() {
  let result = await delay();
  console.log(result);
}

run();
