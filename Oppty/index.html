4. Operators

Learn about operators in JavaScript

What are Operators?

Operators are symbols that perform operations on operands (values and variables). JavaScript has various types of operators for different operations, from simple math to complex logic.

Arithmetic Operators

Used to perform mathematical operations:

let a = 10; let b = 2;
console.log(a + b);  // Addition: 12
console.log(a - b);  // Subtraction: 8
console.log(a * b);  // Multiplication: 20
console.log(a / b);  // Division: 5
console.log(a % b);  // Modulus (Remainder): 0
console.log(a ** b); // Exponentiation: 100
console.log(++a);    // Increment: 11
console.log(--b);    // Decrement: 1

Assignment Operators

Used to assign values to variables:

let x = 10;
x += 5; // Same as x = x + 5 (15)
x -= 2; // Same as x = x - 2 (13)
x *= 3; // Same as x = x * 3 (39)
x /= 3; // Same as x = x / 3 (13)
x %= 4; // Same as x = x % 4 (1)

Comparison Operators

Used to compare values and return a Boolean:

let p = 5; let q = "5";
console.log(p == q);  // Equal (value only): true
console.log(p === q); // Equal (value & type): false
console.log(p != q);  // Not equal: false
console.log(p !== q); // Not equal (value & type): true
console.log(p > 2);   // Greater than: true
console.log(p <= 4);  // Less than or equal: false

Logical Operators

Used to determine the logic between variables or values:

let age = 22; let hasLicense = true;
// AND operator (Both must be true)
console.log(age >= 18 && hasLicense); // true

// OR operator (At least one must be true)
console.log(age < 18 || hasLicense); // true

// NOT operator (Inverts Boolean)
console.log(!hasLicense); // false

Extra: Conditional (Ternary) Operator

A shorthand way to write an if-else statement:

let score = 85;
let result = (score >= 50) ? "Pass" : "Fail";
console.log(result); // "Pass"

Extra: String & Type Operators

Concatenation

The + operator can also join strings together.

let text = "Hello" + " " + "World"; // "Hello World"

typeof Operator

Used to find the data type of a variable.

typeof "John"; // returns "string"
typeof 3.14;   // returns "number"
