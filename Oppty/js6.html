6. JavaScript Objects

Learn how to store related data and functions in one place

What are Objects?

In real life, a car is an object. A car has properties like weight and color, and methods like start and stop.

Key-Value Pairs

JavaScript objects are collections of named values. We call these values properties. You can think of an object as a variable that can contain many different values.

Object Definition

You define a JavaScript object with an object literal using curly braces {}.

const person = {
  firstName: "John",
  lastName: "Doe",
  age: 50,
  eyeColor: "blue"
};

Accessing Object Properties

You can access object properties in two ways: Dot Notation or Bracket Notation.

// Dot Notation (Common)
let name = person.firstName;

// Bracket Notation
let name = person["firstName"];

Object Methods

Objects can also have methods. Methods are actions that can be performed on objects. A method is a property containing a function definition.

const person = {
  firstName: "John",
  lastName : "Doe",
  id       : 5566,
  fullName : function() {
    return this.firstName + " " + this.lastName;
  }
};

The "this" Keyword

In a function definition, this refers to the "owner" of the function. In the example above, this is the person object.
