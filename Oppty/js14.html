14. ES6+ Features

Learn modern JavaScript: let/const, arrow functions, template literals, destructuring, spread & more

What is ES6+?

ES6 (ECMAScript 2015) introduced modern JavaScript features that make code cleaner, shorter, and easier to read. ES6+ means ES6 and later versions (ES7, ES8...)

Most modern JavaScript (React, Node.js, frameworks) uses ES6+ features every day.

let vs const

let allows reassignment. const does not allow reassignment (but objects/arrays can still be updated inside).

let count = 1;
count = 2; // ✅ allowed

const pi = 3.14;
// pi = 3.1415 ❌ error (reassignment not allowed)

const user = { name: "Alex" };
user.name = "John"; // ✅ allowed (object property update)

Arrow Functions

Arrow functions are a shorter way to write functions. They are commonly used in callbacks.

// Normal function
function add(a, b) {
  return a + b;
}

// Arrow function
const add2 = (a, b) => a + b;

console.log(add2(5, 3)); // 8

Template Literals

Template literals use backticks (`) and allow variable embedding using ${ }.

let name = "Nimisha";
let course = "JavaScript";

let msg = `Hi ${name}, welcome to ${course}!`;
console.log(msg);

Destructuring

Destructuring helps extract values from arrays/objects easily.

// Array destructuring
const nums = [10, 20, 30];
const [a, b] = nums;
console.log(a, b); // 10 20

// Object destructuring
const person = { firstName: "John", age: 22 };
const { firstName, age } = person;
console.log(firstName, age);

Spread & Rest Operators

Spread (...)

Spread expands arrays/objects into individual items.

const arr1 = [1, 2];
const arr2 = [...arr1, 3, 4];
console.log(arr2); // [1,2,3,4]

Rest (...)

Rest collects multiple values into an array.

function sum(...nums) {
  return nums.reduce((t, n) => t + n, 0);
}

console.log(sum(1, 2, 3, 4)); // 10

Default Parameters

Default parameters allow you to provide a fallback value if an argument is missing.

function greet(name = "Guest") {
  return `Hello ${name}`;
}

console.log(greet());        // Hello Guest
console.log(greet("Alex"));  // Hello Alex
