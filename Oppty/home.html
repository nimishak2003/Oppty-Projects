What are Operators?

JavaScript operators are symbols used to perform operations on variables and values.

1. Arithmetic Operators

let x = 10;
let y = 3;
console.log(x + y);

2. Assignment Operators

let x = 10;
x += 5;

3. Comparison Operators

console.log(5 > 3);

4. Logical Operators

let age = 25;
if(age > 18 && age < 30){
   console.log("Young Adult");
}
