4. JavaScript Operators

Learn how to perform calculations and logical operations

What are Operators?

In JavaScript, an operator is a special symbol used to perform operations on operands (values and variables). For example, in 5 + 2, the + is an operator and 5 and 2 are operands.

Arithmetic Operators

Arithmetic operators are used to perform common mathematical calculations.

  • + (Addition): Adds values together
  • - (Subtraction): Subtracts one value from another
  • * (Multiplication): Multiplies values
  • / (Division): Divides one value by another
  • % (Modulus): Returns the division remainder
  • ** (Exponentiation): Raises to the power of
let x = 10 + 5;  // 15
let y = 10 % 3;  // 1 (remainder)
let z = 2 ** 3;  // 8 (2 to the power of 3)

Assignment Operators

Assignment operators assign values to JavaScript variables.

let x = 10;
x += 5; // Same as x = x + 5 (Result: 15)
x *= 2; // Same as x = x * 2 (Result: 30)

Comparison Operators

Comparison operators are used in logical statements to determine equality or difference between variables or values.

  • == : Equal to
  • === : Equal value and equal type (Strict)
  • != : Not equal
  • > : Greater than
  • < : Less than

Strict Equality (===)

Always prefer === over ==. The strict operator checks both the value and the data type, preventing unexpected bugs.

Logical Operators

Logical operators are used to determine the logic between variables or values.

let x = 6;
let y = 3;

(x < 10 && y > 1) // true (AND)
(x == 5 || y == 5) // false (OR)
!(x == y)         // true (NOT)
